# React

- React는 사용자 인터페이스를 구축하기 위한 JavaScript 라이브러리.
- 컴포넌트를 조립해서 UI를 만드는 라이브러리.

# JSX JavaScript XML

- xml : 데이터를 저장하고 전송하기 위한 구조를 정의 하는 언어
- JSX : JS 코드 안에서 html과 유사한 문법으로 UI구조를 작성할 수 있는 JS의 확장 문법

```jsx
function App() {
  return (
    <div>
      <h1>Hello React!</h1>
    </div>
  );
}
```
