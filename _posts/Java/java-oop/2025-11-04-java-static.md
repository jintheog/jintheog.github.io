# Static (정적인)

- 클래스는 정의 하는것 (설계도)
- object들은 그 클래스를 기반으로 만들어진 실체화된 객체, 인스턴스라고 함

- 클래스란 것도 결국 어떤 데이터들의 집합 이기 때문에 이 클래스가 어떤 변수를 가질 수 있고 어떠한 기능을 가질 수 있음.
- 그래서 이 클래스 한테 변수 혹은 어떤 메서드를 정하기 위해서 만들어놓은 키워드가 `static` 키워드임
- `static` 키워드가 붙으면 변수 혹은 메서드가 이 클래스에 저장이 된다
  - `static` 키워드가 안붙어있는 모든 변수들과 메서드들은 전부 인스턴스 변수 인스턴스 메서드 이다.

# `static`이 붙는 순간 클래스에 저장 된다

- 그렇기 때문에 상위 클래스의 모든 하위 객체들이 접근 가능 하다.
